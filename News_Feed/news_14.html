საიტების აწყობის წრე

ანდრია რაზმაძის სახელობის ქუთაისის N41 ფიზიკა-მათემატიკის საჯარო სკოლის და Lopus Electronics ის თანამშრომლობის ფარგლებში ფუნქციონირებს იწყებს სასწავლო წრეები კერძოდ: საიტების აწყობის წრე, ლეგო რობოტიკის წრე და არდუინოს და ელექტოინჟინერიის წრე.

საიტების აწყობის წრეზე მოსწავლეების შეისწავლიან საიტების აწყობას, მიიღებენ გამოცდილებას პროგრამირების ენებში როგორიცაა HTML, CSS და JavaScript. აღნიშნული ენების გარდა, მოსწავლეებს შეხება ექნებათ სერვერებთან, რომლებსაც გამოიყენებენ თავიანთი საიტების გასაშვებად. ამ ყველაფერთან ერთად მოსწავლეები შეისწავლიან Visual Studio Code ში და GITHUB ში მუშაობას.

საიტების აწყობის წრე საშუალებას მისცემს მოსწავლეებს ყოველგვარი წინარე გამოცდილების გარეშე კურსის ბოლოს ააწყონ საიტები და იმუშავონ კომპლექსურ პროექტებზე, რაც დაეხმარება მათ მომავალში ამ საქმიანობის გაგრძელებაში.

კურს გაუძღვება საიტების აწყობის პედაგოგი 3 წლიანი გამოცდილებით - დავით ყაველაშვილი.
